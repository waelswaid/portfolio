FriendService dependency injection refactor (5 files):

- friend_service.py: Replaced 6 standalone functions (send_friend_request, friend_request_accept, friend_request_declined, friend_remove, return_friend_list, return_pending_list) with a FriendService class that receives session_factory via __init__. Removed hardwired `from database import async_session`.
- context.py: Added friend_service field to Deps dataclass (required, no default).
- friend_handlers.py: Removed module-level imports of manager and friend_service functions. _send_result uses ctx.deps.manager instead of direct manager import. All handlers access friend_service via ctx.deps.friend_service.
- main.py: Instantiates FriendService(async_session), stores on app.state.friend_service.
- chat_websocket.py: Passes friend_service in Deps construction.
